Tenant Management System

Complete property management solution for landlords and agents

About The Project

This Tenant Management System is a comprehensive web-based solution designed for landlords, property managers, and real estate agents. It streamlines all aspects of property management including tenant tracking, rent collection, maintenance requests, and financial reporting.

The system features automated M-Pesa payment tracking, where tenants can pay rent via STK Push and the system automatically updates their payment status. Property owners can access real-time dashboards showing occupancy rates, expected rent, and payment history.

Key Features

  • Tenant and property registration
  • Automated rent collection with M-Pesa
  • Lease agreement management
  • Maintenance request tracking
  • Payment history and receipts
  • Automated rent reminders via SMS
  • Financial reports and analytics
  • Role-based access (Admin, Landlord, Tenant)

Tech Stack

PHP MySQL JavaScript Bootstrap M-Pesa API Chart.js

Project Duration

6 weeks

Client

Property Management Company (50+ units)

Results

✅ Reduced rent collection time by 70%
✅ 95% on-time rent payments
✅ Automated 200+ payment reminders monthly
